2

Built for developers

Instant CDN

Files served via jsDelivr with edge caching worldwide. Copy-ready URLs, HTML, and Markdown snippets.

Batch upload

Upload multiple files as a single Git commit. Drag entire folders. Paste screenshots with Ctrl+V.

Auto optimization

Images automatically compress to WebP before upload — smaller files, faster loads, zero effort.

Your repo, your data

Files live in your own GitHub repository. No vendor lock-in, no third-party storage. Full Git history.

Self-host option

GitAssets is open source under the MIT license. Fork it and run your own instance on GitHub Pages, with a small Cloudflare Worker for auth.
